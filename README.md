# aidawx
Aida WxApp 产品需求整理

### [必读须知] 项目说明事项

本项目为小程序整体项目的其中一部分。

主要目标是完成以下几点
1. 卡片的加载（通过服务器的API）
2. 卡片的交互（用户可以左右滑动操作卡片，用户还可点数字浮点）
3. 按钮的交互（通过条件控制按钮的显示状态）
4. 详情浮层的展示（及交互）
